# Sales & Customer Analytics Dashboard (Power BI)

Interactive Power BI dashboard built using the Adventure Works dataset to analyze sales performance, customer behavior and product insights.

---

## Overview

This project demonstrates an end-to-end BI solution including data modeling, DAX calculations and interactive dashboard design.  
The report provides clear visibility into key business metrics and supports data-driven decision-making.

---

## Key Features

- KPI tracking: Revenue, Profit, Orders, Return Rate  
- Time intelligence: YTD, rolling metrics, previous month comparison  
- Customer and product performance analysis  
- Geographic analysis using map visualizations  
- Target vs Actual comparison  
- Interactive filters and drill-down functionality  
- What-if parameter for profit simulation  

---

## Data Model

- Star schema design  
- Fact tables: Sales, Returns  
- Dimension tables: Customer, Product, Calendar, Territory
